# To-Do List App ✅

A clean and modern **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
This app helps users manage daily tasks with a simple and professional user interface.

---

# ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Local storage support
- Responsive design
- Modern UI design
- Keyboard Enter support
- Persistent task saving

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
